Double Hashing

Question: You have the following keys that need to be stored for fast retrieval. Use Double Hashing to insert them into a hash table.
Given:
Table Size m = 11
h₁(k) = k mod 11
h₂(k) = 7 - (k mod 7)
Keys to Insert (in order): 23, 45, 67, 12, 34, 56, 78, 89

Hash Table (Size = 11)

45 45 mod 11 = 1 7 - (45 mod 7) = 7 - 3 = 4 i=0: 1 (occupied)
i=1: (1 + 1×4) mod 11 = 5 ✓		 5

i=1: (1 + 1×3) mod 11 = 4 ✓		 4

i=1: (1 + 1×2) mod 11 = 3 ✓		 3
