COLOR MODELS

How does the computer work with colors?

Color Depth

Color depth depends on the number of bits used to represent each pixel, determined by the encoding system.

Common Standard: 8 bits per channel for RGB (8 x 3 = 24 bits total)
24-bit color = 224 = 16,777,216 colors
Bit Depth Colors Available Quality
1 bit 2 Black & White
8 bits 256 Grayscale / Indexed
24 bits 16.7 million True Color

RGB Color Model

RGB = Red, Green, Blue

  • Used for digital displays (monitors, TVs, phones)
  • Values range: 0-255 or 0.0-1.0
  • Additive color system
  • Starts with black, adds light
All colors combined = White
No colors = Black

In OpenGL, colors are specified using values from 0.0 to 1.0:

CMYK Color Model

CMYK = Cyan, Magenta, Yellow, Black (K)

  • Used for printing
  • Subtractive color system
  • Starts with white paper
  • Ink absorbs (subtracts) light

HSV/HSL Color Model

More intuitive way to select colors, based on RGB:

  • Saturation (0-100%) - Color intensity
  • Value/Lightness (0-100%) - Brightness

COLOR HARMONY

Aesthetically pleasing color combinations

What is Color Harmony?

Harmony = A pleasing arrangement of colors

Without Harmony:

  • Colors appear chaotic or boring
  • Visual discomfort for viewers
  • Message gets lost

With Harmony:

  • Colors are pleasing to the eye
  • Creates visual balance
  • Enhances communication

Color harmony is achieved through geometric relationships on the color wheel.

CPIT285 - Understanding Colors

Color Harmony Schemes

Gradual Harmony

  • Monochromatic - One color, different values
  • Analogous - Adjacent colors on wheel

Contrastive Harmony

  • Complementary - Opposite colors
  • Split Complementary - One + two adjacent to opposite
  • Triadic - Three equally spaced
  • Tetradic - Four colors (rectangle/square)

Monochromatic Color Scheme

Uses one color with variations in saturation and value.

  • Easiest scheme to create
  • Always harmonious
  • Clean, elegant look
  • Can lack contrast if not careful
Tip: Use different tints, tones, and shades of the same hue to create depth.

Example: Purple Monochromatic

Analogous Color Scheme

Uses colors adjacent to each other on the color wheel (typically 3-5 colors).

  • Found in nature frequently
  • Harmonious and pleasing
  • Creates a sense of unity
  • Low contrast, comfortable viewing
Example: Blue to Green range creates a cool, natural feel.

Example: Blue-Green Analogous

Complementary Color Scheme

Uses colors opposite each other on the color wheel.

  • Maximum contrast and impact
  • Vibrant and energetic
  • Great for drawing attention
  • Can be overwhelming if overused
Tip: Use different shades and tints to soften the contrast.

Example: Blue & Orange

Split Complementary Color Scheme

Uses one color plus the two colors adjacent to its complement.

  • Strong visual contrast
  • More nuanced than complementary
  • Easier to balance
  • Three-color harmony
Example: Red with Yellow-Green and Blue-Green

Example: Red + Split Complement

Triadic Color Scheme

Uses three colors equally spaced on the color wheel (120° apart).

  • Vibrant and dynamic
  • High visual contrast
  • Creates bold, playful designs
  • Best with one dominant color
Tip: Let one color dominate; use the other two as accents.

Example: Red, Yellow, Blue (Primary Triad)

Think: Super Mario!

Tetradic (Double Complementary) Scheme

Uses four colors forming a rectangle or square on the wheel.

  • Rich and diverse palette
  • Two complementary pairs
  • Requires careful balancing
  • Popular in logos (Google, Microsoft, eBay)
Warning: Equal usage works for logos but needs one dominant color for larger designs.

MEANINGS & FEELINGS OF COLORS

Color Psychology in Design

Warm, Cool, and Neutral Colors

Warm Colors

Red, Orange, Yellow

Exciting, energetic, passionate. Can evoke anger or urgency if overused.

Cool Colors

Blue, Green, Purple

Calming, professional, trustworthy. Can feel cold or impersonal alone.

Neutral Colors

Black, White, Gray, Brown

Balance other colors. Great for backgrounds. Black = shade, White = tint.

Key Takeaways

  • RGB is additive (screens), CMYK is subtractive (print)
  • HSV/HSL provides intuitive color selection (Hue, Saturation, Value)
  • The Color Wheel has 12 colors: 3 primary, 3 secondary, 6 tertiary
  • Tint = color + white, Shade = color + black, Tone = color + gray
  • Color Harmony creates visually pleasing combinations
  • Choose colors that match your intended meaning and feeling
  • Consider warm/cool/neutral balance in your designs
