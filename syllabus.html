B. Course Description & Objectives

Course Description

The course focuses on understanding web application concepts and techniques for managing, controlling, and improving security. It describes the concept of Web Application Security (WAS) applied in different organizations to develop and deploy secure web applications. This course exposes students to contemporary knowledge and techniques of WAS, enabling them to apply these tools and practices to develop and deploy secure web applications. In addition, it enables them to identify and handle various types of vulnerabilities and attacks that threaten web applications.

Course Main Objectives

To comprehend the following knowledge:

  • Web Technologies: HTTP protocol, coding designs, architectures, Web applications, AJAX, XML, JSON technologies/languages.
  • Security Principles: Principles of securing common areas of functionality of web applications.
  • Defense: Defend the integrity of your databases, file systems, user accounts, and many other important resources from various types of vulnerabilities and attacks.
  • Vulnerability Handling: Identify and handle various types of vulnerabilities and attacks that threaten web applications.
  • Secure Development: Apply learned concepts using secure development and deployment methodologies.

D. Course Content (10 Topics)

Topic 1 Introduction & History of Software Security 75 min + Lab
  • Course introduction and objectives
  • The origins of hacking: Enigma Machine (1930s)
  • Code breaking and the "Bombe" (1940s)
  • Telephone phreaking and blue boxes (1950s-1960s)
  • The origins of computer hacking (1980s)
  • Rise of the World Wide Web and web attacks (2000s)
  • Hackers in the modern era (2015+)
  • Introduction to modern web application architecture
Topic 2 Web Applications Technologies 75 min + Lab
  • HTTP Protocol fundamentals
  • Modern vs Legacy web applications
  • REST APIs and API architecture
  • AJAX technologies
  • XML data format and processing
  • JSON technologies and languages
  • Web servers and databases
Topic 3 Web Application Security Fundamentals I 75 min + Lab
  • Server firewall concepts
  • Network Security basics
  • Application Security overview
  • Injection attacks
  • Cross-Site Scripting (XSS)
  • Cross-Site Request Forgery (CSRF)
  • Security Misconfiguration
Topic 4 Web Application Security Fundamentals II 75 min + Lab
  • Input Validation techniques
  • Defense-in-Depth Approach
  • Classifying and Prioritizing Threats
  • Vulnerability Scoring System (CVSS)
Topic 5 Web Application Security Principles I 75 min + Lab
  • Authentication mechanisms
  • Access Control models
  • Authorization frameworks
  • Session management
Topic 6 Web Application Security Principles II 75 min + Lab
  • Browser Security features
  • Database Security practices
  • File Security and upload handling
  • Content Security Policy (CSP)
Topic 7 Secure Development Methodologies I 75 min + Lab
  • Penetrate-and-Patch approach
  • Holistic security approach
  • Threat Modeling techniques
  • Secure Coding Libraries
  • Security Testing methodologies
Topic 8 Secure Development Methodologies II 75 min + Lab
  • Secure Application Architecture
  • Vulnerability Management processes
  • Security code review
  • Bug bounty programs
Topic 9 Secure Development Methodologies III 75 min + Lab
  • Defending Against XSS attacks
  • Defending Against CSRF attacks
  • Defending Against XXE attacks
  • Input sanitization techniques
Topic 10 Secure Development Methodologies IV 75 min + Lab
  • Defending Against Injection attacks
  • Defending Against DoS attacks
  • Securing Third-Party Dependencies
