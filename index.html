Course Overview

The course focuses on understanding web application concepts and techniques for managing, controlling, and improving security. It describes the concept of Web Application Security (WAS) applied in different organizations to develop and deploy secure web applications. This course exposes students to contemporary knowledge and techniques of WAS, enabling them to apply these tools and practices to develop and deploy secure web applications.

Course Learning Outcomes (CLOs)

1. Knowledge and Understanding
  • CLO 1.1 Identifying basic web technologies concepts, techniques, and communications.
  • CLO 1.2 Defining various types of vulnerabilities on web applications.
  • CLO 1.3 Listing various types of attacks on web applications.
  • CLO 1.4 Identifying various types of techniques and methodologies to secure the development and deployment of web applications.
2. Skills
  • CLO 2.1 Estimating various types of vulnerabilities on web applications.
  • CLO 2.2 Detecting various types of attacks on web applications.
  • CLO 2.3 Executing various types of techniques and methodologies to secure the development and deployment of web applications.
3. Values
  • CLO 3.1 Building and managing secure web applications.
  • CLO 3.2 Facilitating secure communication using web applications by minimizing the threats.
